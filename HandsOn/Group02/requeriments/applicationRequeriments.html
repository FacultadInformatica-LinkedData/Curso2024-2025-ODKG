App Description

Overview

This mobile application is designed to facilitate the search and location of bicycle parking stations in urban environments, using open data to provide accessible and useful information to cyclists.

Key Features:

1. Bicycle Parking Search: Users can enter their current location or explore the map to visualize the nearest bicycle parking stations. The application utilizes an intuitive interface that allows users to quickly find the most convenient option.

2. Detailed Information: By selecting a bicycle parking station, users will access relevant information.

Interface:

The user can search for bicycle parking stations based on various criteria, including the current location (street or neighborhood), the type of parking station (closed, open, etc.) and other specific features. The user has a map where they can find the bicycle parking stations located near them.
