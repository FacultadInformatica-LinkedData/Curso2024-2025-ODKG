Analysis of Public Electric Vehicle Charging Points in Madrid

Data Analysis:

The following entities have been identified from the analysis of the **Public Electric Vehicle Charging Points** dataset:
Entity Description
ID_Punto Unique identifier for each charging station. (e.g., ID_Punto like "04 VP").
Distrito District in Madrid where the charging station is located. (e.g., DISTRITO like "Arganzuela").
Barrio Neighborhood in Madrid where the charging station is located. (e.g., BARRIO like "Palos de Moguer").
Estado Operational status of the charging point (e.g., ESTADO like "Ejecutado").
Ubicación Address of the point where the vehicle can be charged (e.g., Ubicación like "Ronda de Valencia 1").
Emplazamiento Type of place where it is found. (e.g., Emplazamiento like "Vía pública").
Info_Adicional More relevant information about the place, if it is necessary. (e.g., Info_Adicional like "Aparcamiento privado rotación").
Nº_Equipos Number of vehicles which can be charged at the same time. (e.g., Nº_Equipos like "1").
CARACTERÍSTICAS_EQUIPO The type of charging available and the equipment characteristics. (e.g., CARACTERÍSTICAS_EQUIPO like "50 kW 2 conectores CC").
Operator Company or entity that operates the charging point (e.g., OPERADOR like "GIC" or "REPSOL").
Gestión Type of arrangement for the electrical vehicle station. (e.g., Gestión like "Convenio vía pública").
INFO_FECHA_APERTURA The date of the day that the station was opened. (e.g, INFO_FECHA_APERTURA like "01/09/2019").
FECHA_APERTURA The date and time when the station was opened. (e.g., FECHA_APERTURA like "05/09/2019 22:00:00").
FECHA_ALTA Date and time when the station was registered. (e.g., FECHA_ALTA like "02/03/2022 15:21:24").
FECHA_EDICION Last time when the information was edited. (e.g., FECHA_EDICION like "27/04/2022 10:43:52").
COD_DIS The code of the district where it is located the station. (e.g., COD_DIS like "2").
COD_BAR The code of the neighborhood where it is located the station. (e.g., COD_BAR like "6").
HORARIO Number of hours that the station is opened. (e.g., HORARIO like "24" or "de 7:30h a 21:00h").
