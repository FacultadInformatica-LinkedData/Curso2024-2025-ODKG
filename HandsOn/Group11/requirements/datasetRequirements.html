Dataset Requirements Analysis

Requirement R1: Data in the Smart Cities Domain

Status: Satisfied

The dataset is related to smart cities, specifically urban mobility and electric car charging stations in Madrid.

Requirement R2: Data are svailable as a CSV File

Status: Satisfied

Data available in CSV and SHP format.

Requirement R3: Open License for Publication

Status: Satisfied

The dataset is licensed under a Creative Commons Attribution 4.0 International License (CC BY 4.0), which allows for open access, redistribution, and publication. This license ensures that users can freely use and share the dataset, as long as proper attribution is given to the source.

Requirement R4: Easily Linked with Real-World Entities

Status: Satisfied

Each station has an ID and locations within streets and districts from Madrid.

Requirement R5 (Optional): Documentation Exists

Status: Satisfied

There are some documents in the webpage that add information to the dataset by explaining the format of the data.

Requirement R6 (Optional): Data from Multiple Sources

Status: Partially Satisfied

The dataset comes from a single source but could be linked with other datasets related to transportation, energy consumption, or urban planning to understand electric cars mobility.
