Description

The ontology is structured using established semantic web standards, including RDF (Resource Description Framework), OWL (Web Ontology Language), and RDFS (RDF Schema).

Key Classes and Properties

  • AirQualityMetric: Represents the measured value of a specific pollutant.
    • Properties: numericValue, measurementUnit, resultTime.
  • Station: Represents the physical location of an air quality monitoring station.
    • Properties: latitude, longitude, altitude, address.
  • Sensor: Represents the individual sensors deployed at a station.
    • Properties: sensorType, observes, madeBySensor.

The relationships between these entities are defined using object properties, facilitating queries about air quality data in relation to time, location, and sensor performance.

This ontology enables users to capture, query, and analyze air quality data effectively, enhancing understanding of air pollution trends and facilitating informed decision-making.
