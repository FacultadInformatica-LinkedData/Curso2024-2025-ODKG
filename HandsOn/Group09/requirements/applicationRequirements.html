Main goal

We aim to show the measurements of air quality within a spatial context, using a Madrid City map and taking advantage of the information that the selected datasets offer:

  • Air quality metrics: SOX, NOX, COX, O3, BTX, PM10, PM2.5
  • Location of measurement stations
  • Date of data collection

Interface requirements

In terms of location
As a user's choice

A user should be able to select to see data from all stations or from only one station.

As a dataset constraint

Not all stations gather data about all parameters. Therefore, if a user selects a parameter whose data is not collected by a given station, that station should not show up on the map.

In terms of time
As a user's choice

A user should be able to select whether data should belong to a specific day (showing that day's value) or to a time window (showing the average value for that period)

In terms of air quality parameters
As a user's choice

A user should be able to select to see data pertaining to a specific parameter, but only one parameter at a time.
