Analysis Part Results
Analyse Data Set ID Cámara: int identifier of the item | Tipo: String describing the type of camera can be FIJA (specific zone) or DOMO (with PTZ control) | Ubicación: String corresponding to the name of the neighbourhood | Resolución: String following resolution format containing two number, it's cuantitative | Zoom Optico: String referring to the amount of zoom x+Int | Año Adquisición: Int, year of adquisition of the camera.
Analyse Licensing of the Data Source The publisher is the "Ayuntamiento de Madrid" and the responsable of the dataset is the "Dirección General de la Policía Municipal"
Define Resource Naming Strategy The naming strategy will follow the next naming "http://CamaraMadrid.org/videocameras/{camera_id}" for example "http://CamaraMadrid.org/videocamera/126", uses "/" instead of "#" because is not a large dataset and it's not updated constantly
