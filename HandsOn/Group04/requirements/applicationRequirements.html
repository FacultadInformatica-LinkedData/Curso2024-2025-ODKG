Application Requirements for "Smart Educational Centers Navigator"

Application Overview

The application aims to provide citizens with a tool to locate educational centers in Madrid based on proximity, transportation options, and accessibility. The application will also allow users to filter educational centers based on the type of education provided (e.g., preschool, high school).

Functional Requirements

  • Search by Location: The user can search for educational centers near their location or in a selected neighborhood.
  • Filter by Education Level: Users can filter results by the type of education offered by each institution.
  • Accessibility Filter: Users can select centers with accessibility options for people with disabilities.
  • Detailed Information Pages: Each center will have its own page displaying detailed information, including contact info, equipment, and transportation options.

Mock-Ups

Home Page

The home page will feature a map of Madrid with markers indicating the locations of educational centers. A search bar and filter options will be provided to help users refine the results.

Search Results

After performing a search, users will see a list of matching educational centers with brief descriptions and distances from their location. Clicking on a center will display more detailed information.

Educational Center Detail Page

This page will show detailed information about a selected educational center, including its description, courses offered, accessibility, and public transportation options. A map with the center's location will also be displayed.
