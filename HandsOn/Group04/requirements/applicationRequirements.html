Application Requirements for "Electric Vehicle Charging Stations Navigator"

Application Overview

The application aims to provide citizens with a tool to locate electric vehicle charging stations in Madrid based on proximity, station capacity, and availability of specific charging connectors. Users will also be able to filter stations based on operator, power capacity, and charging type (e.g., CCS, CHAdeMO).

Functional Requirements

  • Search by Location: The user can search for charging stations near their current location or in a specific neighborhood.
  • Filter by Charging Capacity: Users can filter results by the charging power (e.g., 50 kW, 100 kW) and connector type (e.g., CCS, CHAdeMO).
  • Station Operator Filter: Users can select stations operated by a preferred operator (e.g., Repsol, Iberdrola, Endesa).
  • Detailed Information Pages: Each charging station will have its own page displaying detailed information, including the number of charging points, location, operator, and available connectors.

Mock-Ups

Home Page

The home page will feature a map of Madrid with markers indicating the locations of electric vehicle charging stations. A search bar and filter options will be provided to help users refine the results.

Search Results

After performing a search, users will see a list of matching charging stations with brief descriptions, including the operator, charging power, and distance from their current location. Clicking on a station will display more detailed information.

Charging Station Detail Page

This page will show detailed information about a selected charging station, including its operator, location, number of available charging points, supported connectors, and charging power. A map with the station's location will also be displayed.
