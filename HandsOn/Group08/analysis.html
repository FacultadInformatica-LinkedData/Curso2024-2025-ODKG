Data Analysis

2024_Accidentalidad.csv

Name Type Description
num_expediente String Unique identifier for the accident
fecha DateTime Contains the date of the accident
hora DateTime Contains the hour of the accident
localizacion String Address where the accident took place
numero String Number of the street (when necessary).
cod_distrito Integer Identifier for the district
distrito String Name of the district.
tipo_accidente Integer Accident description
estado_meteorologico String Weather description
tipo_vehiculo String Vehicle description
tipo_persona Float Role of the person in the accident: pedestrian, driver, passenger...
rango_edad String Age range of the person
sexo String Sex of the person
cod_lesividad Integer Identifier for the severity of the lesions caused by the accident
lesividad String Description of the sanitary action taken
coordenada_x_utm Float Location x coordinate
coordenada_y_utm Float Location y coordinate
positiva_alcohol String Identifies if the person was tested positive for alcohol
positiva_droga String Identifies if the person was tested positive for drugs

COD_DIS Integer Unique identifier of the district
COD_DIS_TX Integer Identifier of the district
DISTRI_MAY String Name of the district in uppercase
NOMBRE String Name of the district
