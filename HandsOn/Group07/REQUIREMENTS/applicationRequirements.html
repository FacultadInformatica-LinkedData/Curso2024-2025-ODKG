Application Requirements

This document outlines the application requirements, including a description of the user interface and its functionality.

Application Overview

The application will display a real-time map showing the events in Madrid in different districts across the city. Users will be able filter acording to date, location, and audience. The goal is to promote culture and making it more accessible by creating a simple and intuitive search system.

Main Features

  • Interactive map displaying events information in Madrid
  • Search functionality to find events near specific locations
  • Filters to show the events on specific dates and their duration
  • Filter according to target audience
  • interactive information of events
