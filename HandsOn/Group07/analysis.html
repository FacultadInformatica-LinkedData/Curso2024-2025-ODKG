Madrid Events Data Analysis

1. Dataset Overview

The dataset contains records of events and activities taking place in Madrid.

Variable Type Description
ID-EVENTOint64Identifier for each event
TITULOobjectName of the event
PRECIOint64Information about the price of attending the event
GRATUITOboolIndicates if the event is free or not
LARGA-DURACIONint64Indicates whether the event repeats over time
DIAS-SEMANAobjectSpecifies which days of the week the event occurs
DIAS-EXCLUIDOSobjectLists days that the event does not occur
FECHAobjectThe start date of the event
FECHA-FINobjectThe end date of the event
HORAobjectThe time when the event starts

3. Resource Naming Strategy

The following resource naming strategy was applied to the dataset for consistency and clarity in Linked Data and RDF representations:

  • Base URI: http://madrid.events.com/
  • Events: http://madrid.events.com/event/{ID-EVENTO}
  • Activities: http://madrid.events.com/activity/{ID-EVENTO}-{TITULO-ACTIVIDAD}
  • Installations: http://madrid.events.com/installation/{NOMBRE-INSTALACION}
  • URLs for Further Info: Link directly using CONTENT-URL, URL-ACTIVIDAD, and URL-INSTALACION.

This strategy ensures that each resource is uniquely identified and can be referenced easily in a semantic web context.
