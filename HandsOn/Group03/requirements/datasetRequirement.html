Group 03: Datasets Requirements

R1: Data are in one of the smart city domains.

Our datasets are about Madrid's housing pricing and about building efficiency. Districts, neigborhoods and building are mappable and have linking potential.

R2. Data are available as a CSV file.

This dataset is available in .CSV, as well as in other formats as XML or JSON, among others.

R3. Data have an open license so they can be published.

This datasets have an open license and allows it to be reused, as described in the following link

R4. Data can be easily linked with generic real-world entities

Has described in the first point, our dataset not only contains basic information about the center, but also location and public transportation information. This allows us to link it with other datasets to enrich our future knowledge graph.

R5. Documentation exists for the data.

There is available a pdf file describing the dataset structure.

R6. Data can come from multiple data sources.

As a future plan, we would like to find more datasets to create new links with our main dataset. As an example, we could create links with the public transportation timetables, or with the main services available in the location of the center.
